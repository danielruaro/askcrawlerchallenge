# AskSuite Nodejs Web Crawler

How to run Crawler

## Install ⚙️

```
npm install
```

## Start 🏃
```
npm run start
```

## Test

```
curl -i -X POST -H "Content-Type: application/json" -d "{\"checkIn\":\"31/12/2019\", \"checkOut\" : \"01/01/2019\"}" http://localhost:3000/rooms
```


## Built with
* Moment
* Puppeteer
* Express
* Joi
